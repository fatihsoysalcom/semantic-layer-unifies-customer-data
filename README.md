# Semantic Layer Unifies Customer Data

This example demonstrates the concept of a semantic layer in AI projects. It simulates data coming from two different 'silos' (Sales and Marketing) with inconsistent identifiers and schemas. A `build_semantic_customer_profile` function acts as a simplified semantic layer, unifying this disparate data into a consistent customer profile. Finally, an 'AI-like' function `analyze_customer_value` shows how AI models can effectively leverage this semantically unified data for richer analysis, which would be difficult or impossible with raw, siloed data.

## Language

`python`

## How to Run

Save the code as `main.py`.
Run from your terminal: `python main.py`
